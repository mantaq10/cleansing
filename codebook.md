#Run Analysis Code Book

This codebook accompanies the file "run_analysis.r" that prepares a tidy data set from the <a href="http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones">UCI HAR data Set</a> and also summarizes the data around the mean value of all means and standard deviations across all activity types of each subject. 
The resulting dataset has 35 rows and 68 columns of tidy data. 

##Key Qualifiers (Groups)

Subject  - Unique identifier of Subject (Total 30 subjects)
Activity - One of Six possible activities

Types of Activities:
 - WALKING
 - LAYING
 - SITTING
 - STANDING
 - WALKING DOWNSTAIRS
 - WALKING UPSTAIRS
